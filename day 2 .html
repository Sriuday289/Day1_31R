FRONT-END TECHNOLOGIES

HTML

HTML stands for Hyper Text Markup Language
It is used to create the structure of a website.

CSS

CSS stands for Cascading Style Sheets
It is a language used to style and format the appearance of web pages, including layout, colors, fonts, and spacing. It works alongside HTML to make web pages visually appealing and responsive across different screen sizes and devices.

JAVASCRIPT

It is a programming language used to create interactive and dynamic elements on websites, such as animations, forms, and real-time updates.
It runs in the browser and allows developers to control website behavior and respond to user actions without needing to reload the page.

BACKEND TECHNOLOGIES

PYTHON

Python is a beginner-friendly programming language known for its easy-to-read syntax and simple structure.
It can be used for various applications like web development, data analysis, automation, and machine learning.

JAVA

Java is a powerful, object-oriented programming language used for building large-scale applications and software.
It is platform-independent, meaning code written in Java can run on any device that has the Java Virtual Machine.

DATABASES

SQL

It is a language used to manage and manipulate data stored in relational databases.
It allows you to perform tasks like retrieving, inserting, updating, and deleting data in tables, making it essential for working with databases.
